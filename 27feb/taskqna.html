JavaScript Basics

1. Introduction to JavaScript

JavaScript is a lightweight, interpreted programming language used to create dynamic and interactive web pages.

2. Ways to Apply JavaScript

Inline JavaScript

<button onclick="alert('Hello!')">Click Me</button>

Internal JavaScript

<script>
    document.write("Welcome to JavaScript!");
</script>

External JavaScript

<script src="script.js"></script>

3. Variables in JavaScript

Variables in JavaScript are used to store data values. They can be declared using var, let, or const.

var name = "Alice"; // Can be redeclared
let age = 25; // Block-scoped
const country = "USA"; // Immutable

4. Introduction to Data Types

  • String: "Hello"
  • Number: 100, 3.14
  • Boolean: true, false
  • Undefined: A declared variable without a value
  • Null: A variable with an empty value
  • Symbol: Unique and immutable values
  • BigInt: For very large numbers
  • Object: {name: "John", age: 25}
  • Array: ["apple", "banana", "mango"]
